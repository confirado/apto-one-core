Achtung:

Das Umbenennen eines Ordners kann zu Fehlfunktionen des Konfigurators führen. Bilder die in Content Snippets, benutzerdefinierten Eigenschaften oder in Element Definitionen hinterlegt sind, werden nicht automatisch angepasst. Sollten Sie eine Fehlfunktion des Konfigurators nach dem Umbenennen bemerken, so können Sie dem Ordner wieder den ursprünglichen Namen geben und die Funktionalität wieder herstellen.
